## 🚨 Problem Statement
During natural disasters, people face three critical challenges:
- They don't know **how severe the risk** is near their location
- They don't know **where the nearest hospital, shelter or police** station is
- They don't know **what safety steps** to take immediately

**SafeZone AI solves all three — in real time, powered by AI.**

## 🔑 Setup & Run Locally

1. Download or clone this repo
2. Open `script.js` and replace:
   - `YOUR_GEMINI_API_KEY` → [Get free from AI Studio](https://aistudio.google.com)
   - `YOUR_OPENWEATHER_API_KEY` → [Get free from OpenWeather](https://openweathermap.org/api)
3. Double-click `index.html` — opens in browser instantly ✅

> Works without API keys too — smart fallback data included

safezone-ai/
├── index.html   → Landing page + full dashboard layout
├── style.css    → Dark theme, glassmorphism, animations, responsive
├── script.js    → Location detection, weather API, risk engine, Gemini chatbot
